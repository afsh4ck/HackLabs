API1:2023 – BOLA (Broken Object Level Authorization)

OWASP API Security Top 10 · Riesgo: CRITICAL

Descripción

BOLA permite acceder a objetos de otros usuarios manipulando IDs en las peticiones API:

  • Cambiar IDs numéricos para acceder a datos de otros usuarios
  • Manipular UUIDs para evadir controles de acceso
  • Escalada horizontal de privilegios

API de Usuarios

Endpoint vulnerable — intenta acceder a usuarios diferentes:

GET /api/bola/users/
